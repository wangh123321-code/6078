## 1. 问题概述

### 原有问题
1. **浏览器崩溃卡顿**: 五代系谱图200+节点时，直接卡成PPT，滚动一次等5秒，有时直接崩溃
2. **无限递归**: 近亲繁殖导致血缘关系形成环，递归遍历无限循环
3. **重复渲染**: 相同节点重复创建多次，没有缓存复用
4. **布局混乱**: 力导向布局节点多了抖来抖去，收敛慢
5. **近亲繁殖无标记**: 血缘关系不清晰，繁育决策困难

### 性能指标要求
| 指标 | 要求 |
|------|------|
| 1000节点首次渲染时间 | < 1秒 |
| 滚动帧率 | ≥ 60 FPS |
| 缩放操作 | 流畅不卡顿 |
| 循环引用处理 | 不会崩溃，能正确标记 |

## 2. 优化方案

### 2.1 后端优化 - CatService.java

**原方案**: 深度优先递归(DFS)，无环检测，无缓存
```java
// 原代码 - 有问题的递归实现
private PedigreeNode buildPedigreeTree(Cat cat, int currentGen, int maxGen) {
    // 无环检测，遇到环会无限递归
    // 无缓存，相同节点重复创建
    if (currentGen < maxGen) {
        if (cat.getFatherId() != null) {
            catRepository.findById(cat.getFatherId()).ifPresent(father ->
                    node.setFather(buildPedigreeTree(father, currentGen + 1, maxGen))
            );
        }
    }
}
```

**新方案**: 广度优先迭代(BFS) + 环检测 + 节点缓存

### 2.2 前端优化

#### 2.2.1 遍历算法优化 - pedigreeLayout.js
```javascript
export function traversePedigreeBFS(root, maxGeneration = 5) {
  const visited = new Set();
  const queue = [{ node: root, level: 0, path: [] }];
  
  while (queue.length > 0) {
    const { node, level, path } = queue.shift();
    
    // 环检测
    if (path.includes(catNo)) {
      cycles.push({ description: `环: ${path.join(' → ')}` });
      continue;
    }
    
    // 已访问节点跳过
    if (visited.has(catNo)) continue;
    visited.add(catNo);
    
    // BFS遍历父节点
    queue.push({ node: node.father, level: level + 1, path: newPath });
    queue.push({ node: node.mother, level: level + 1, path: newPath });
  }
}
```

#### 2.2.2 虚拟滚动实现
```javascript
export function getVisibleNodes(nodes, viewport) {
  const { scrollX, scrollY, viewWidth, viewHeight } = viewport;
  const margin = 100; // 预渲染缓冲区
  
  return nodes.filter(node => {
    // 只渲染视口范围内的节点（含缓冲区）
    return node.x + node.width > scrollX - margin &&
           node.x < scrollX + viewWidth + margin &&
           node.y + node.height > scrollY - margin &&
           node.y < scrollY + viewHeight + margin;
  });
}
```

#### 2.2.4 节点缓存复用
```javascript
export class NodeCache {
  private cache = new Map();
  
  getOrCreate(catNo: string, creator: () => PedigreeNode) {
    const existing = this.cache.get(catNo);
    if (existing) {
      this.hitCount++;
      return existing;
    }
    const newNode = creator();
    this.cache.set(catNo, newNode);
    this.missCount++;
    return newNode;
  }
}
```
